Stockchart Demo Page

Send your ReST form to the following address: 

http://stockchartaws-env.nkp696mepp.us-east-1.elasticbeanstalk.com/addmulti

Format a JSON message as a PUT request: 


{
"stocks" : [
	{
	"stock":"A2M",
	"dateAdded" : "2018-10-20T13:23:43.533Z",
	"price" : 8.44,
	"numberOfUnits": 15
	},
	{
	"stock":"ABC",
	"dateAdded" : "2018-10-20T13:23:43.533Z",
	"price" : 2.43,
	"numberOfUnits": 22
	},
	{
	"stock":"AGL",
	"dateAdded" : "2018-10-20T13:23:43.533Z",
	"price" : 7.13,
	"numberOfUnits": 10
	},
	{
	"stock":"ALL",
	"dateAdded" : "2018-03-11T13:23:43.533Z",
	"price" : 24.44,
	"numberOfUnits": 1
	}
	],
	"comments" : "stockchart@sharklasers.com"
}

Notes:

The comments element is an email address for returned PDF. The email address has to be pre-registered with Amazon SES.
This email address can be used for testing: stockchart@sharklasers.com
The inbox can be viewed by browsing to https://www.guerrillamail.com and choosing the 'stockchart' inbox.

There is a limit of 10 stocks that can be added for a single PDF page report.
